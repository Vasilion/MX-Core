Setup
  - Clone Repo
  - Install Node
  - cd to project directoy and run 'npm i'

Run + Debug
  - Install the 'Ionic' extension in vs code
  - 'Alt + D' will launch the app in debug mode
  - Open chrome dev tools and change view to Iphone or Samsung
